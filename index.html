Desarrolló Web

Web es una palabra inglesa que significa red o telaraña. se designa como "la Web" al sistema de gestion de informacion mas popular para la transmision de datos a traves de internet. La Web es el diminutivo de World Wide Web o wwww cuyas tecnologias para su funcionamiento (HTML,URL,HTTP) fueron desarroladas en el año 1990 por Tim Berners Lee.

Un Poco de Historia de la Web

La historia de la web abarca ya mas de 30 añs, en los que se han alternado periodos de intenso Desarrollo con otros de estancamiento. El primer servidor de paginas web de la historia se puso en marcha en diciembre de 1990 en el CERN(Ginebra Zuiza). El inventor de la Web, el informatico ingles Tim Berners-Lee, pretedia crear un sistema que permitiera a los investigadores del CERN compartir facilmente la informacion. La primera version del lenguaje de marcas inventado por Berners-Lee nunca fue publicado como documento oficial, pero si lo hubiera sido hubiera llamado HTML 1.0. Los investigadores del CERN, precedentes de todas partes de el mundo, diseminaron en sus universidades de origen el sistema creado por Berners-Lee, puesto que se trataba de un sistema abierto y libre. En aquella epoca ya existia el internet, pero su acceso estaba limitado principalmente a Universidades y Centros de investigacion. En noviembre de 1993 se publico la version 1.0 de Mosaic, un navegador creado en la universidad de illinois por Marc Andeessen y se superaba a todos al permitir, por ejemplo, incluir imagenes en las paginas Web. En 1994 se permitio el accceso de particulares y empresas a internet. La Web se convirtio enseguida en el servicio mas empleado para ofrecer informacion, eclipasando a servicios competidores como Gopher o WAIS. La Web empezo a verse como una gigantesca oportunidad de negocio y Marc Andreessen dejo la universidad para fundar Netscape, que publicaria la version 1.0 de su navegador en diciembre de 1994. Para dirigir el desarrollo de la Web, Berners-Lee fundo el Worl Wide Web Consortium (W3C) en octubre de 1994, como lugar de encuentro de empresas, universidades y organizaciones sin animo de lucro. El W3C esta organizado en grupos de trabajo(WG,Working Groups). Los primeros Grupias de trabajo que se crearon se dedicaron al HTML y a las CSS.

En 1995 Microsoft Incluyo en Windows un navegador, Internet Explorer, que poco a poco comenzo a ganar cuota de mercadp a costa de Netscape. Daba comienzo a la llamada guerra de los anvegadores. Entre 1995 y 2000 Microsoft y Netscape compitieron a un ritmo frenetico para la epoca, publicando nuevas versiones cada año. Para diferenciar sus productos, cada navegador fue incorporando nuevas etiquetas, lo que supuso un riesgo de fragmentacion de la Web. En estos años, el W3C tambien publico recomendaciones a ritmo frenetico. Pro un lado, para consensuar un HTML comun para todos los navegadores Pero por otro lado , proponiendo inovaciones muy importantes, como la separacion entre contenido y presentacion mediante hojas de estilo(CSS). En 1995 Brian Eitch creo para Netscape 2.0 el lenguaje de programacion Javascript, cuyos programas se odian incluir directamente en las paginas Web para ser ejecutadas por el navegador. Microsoft creo su propia variante parcialmente incompatible. La normalizacion de Javascript no la llevo acabo el W3C, si no la organizacion ECMA, que en 1997 em pezo a publicar normas para unificar y desarrollar el lenguaje, bajo el nombre de ECMAScript.

  • Visibilidad de contenidos
  • actualizacion de contenidos
  • Actividad o fin Principal con respecto a la visibilidad de contenidos debemos estructurar los sitios web en:
  • Intranets Una Intranet es un sitio web instaladoen una red privada con un fin principalmente empresarial u organizacional.
  • Extranets La extranet se extiende como una "intranet llevada a internet", es decir sigue siendo un sitio Web con una finalidad privada, restringida a un numero determinado de personas, pero cuyo alcance requiere que este en internet. como ejemplos de extranets podemos considerar cualquier area privada de un sitio Web que permite al usuario consultar, modificar,crear y eliminar datos externos de la empresa.

  • Sitios Web que combinan ambos tipos En el sector de las Webs de tipo empresarial, suele ser habitual combinar cierta informacion estatica(Quienes somos, Donde estamos, filosofia,etc.) con informacion dinamica(Noticias de la empresa)Podemos incluir aqui periodicos, informacion del tiempo, bolsa, etc.

Hoy en día, un sitio web representa el medio más moderno de publicación y comunicación. La creación de una página web requiere el uso de un lenguaje específico, llamado HTML (Hyper Text Markup Language). HTML es un lenguaje de “marcado” que se utiliza para la creación, representación y estructuración del contenido de una página web mediante elementos de formato o etiquetas.Estas etiquetas tienen significados bien definidos, como insertar una imagen, crear un enlace o un párrafo, etc., y generalmente se insertan en el texto de un documento antes de que el navegador web las intérprete y muestre. HTML es un estándar publicado por W3C (World Wide Web Consortium)

Sintaxis HTML

El lenguaje HTML fue creado para estructurar y dar sentido al contenido de la página web. Su sintaxis gira esencialmente en torno a tres términos básicos: elemento, etiqueta y atributo.

Elementos

Los elementos se refieren a los identificadores que describen objetos (texto, imagen u objeto multimedia) dentro de una página web, incluida la estructura y el contenido. Los elementos estan formados por una etiqueta de apertura, contenido y una etiqueta de cierre.

Etiquetas

Para comprender a mayores rasgos los que son las etiquetas HTML te mostramos cuáles son sus partes básicas y para qué sirve cada una:

Elemento: es el nombre de la etiqueta, generalmente pareada (es decir: con apertura y cierre, como lo acabamos de mencionar), y aparece entre los signos de apertura. Por ejemplo: Partes básicas de etiquetas HTML, que indica que esa frase es un título.

Atributo: es lo que diferencia a un elemento y que le da ciertas características, como tamaño, color, tipo de fuente. Se escribe antes del signo de igual (=), junto al elemento de la etiqueta, únicamente en la apertura, y antes de la variable. Por ejemplo: . Este texto aparecerá en tipografía Arial. El atributo es el tipo letra, es decir: face.

Contenido: s el texto que modifica las etiquetas, lo que aparece entre las etiquetas y atributos.

Variable: es la característica del atributo de una etiqueta: color, tamaño, tipo de letra, y se escribe generalmente entrecomillado después del signo de igual. En el ejemplo de atributo, la variable es Arial.

Etiquetas emparejadas: formadas por dos etiquetas, una de apertura y otra de cierre. La etiqueta de apertura está marcada al principio con el signo "<" y al final con el signo “>”. La etiqueta de cierre sigue el mismo principio, pero el nombre del elemento está precedido por el carácter de diagonal "/".

Ejemplo.-
Etiqueta para definir texto importante
<strong> mostrar un texto en negrita</strong >

Etiquetas singulares: Estas etiquetas se utilizan con mayor frecuencia para insertar un elemento proveniente de un recurso externo en en una ubicación bien definida en la página web. Para este tipo de etiqueta, no es necesario marcar el inicio y final de este elemento.

Ejemplo .-
Etiqueta para agregar una imagen
< img src="http:/logo.jpg" alt="Logo"/ >

Nota.- Para no confundir los conceptos de “etiqueta” y “elemento”, consideremos el siguiente ejemplo:
<p> Este elemento contiene texto normal y términos en <b> negritas </b> </p>

Además del texto, el elemento de parrafo <p> contiene otras etiquetas (en este caso, < b > como ejemplo), mientras que la etiqueta <b > solo contiene texto. Notamos que el contenido de un elemento pueden ser otro elemento con sus respectivas etiquetas.

Atributos

Los atributos son propiedades que se utilizan para agregar declaraciones adicionales a un elemento dado (un identificador (id), una fuente (src), etc.). Siempre están incrustados en la etiqueta de apertura y deben estar separados entre sí por al menos un espacio. Cada atributo generalmente acepta un valor entre comillas. Un elemento con atributos puede aceptar la siguiente sintaxis:

< nombre_etiqueta atributo="valor" > Contenido del elemento < /nombre_etiqueta >

Ejemplo.-
< video src="video/vacaciones.mp4" > < /video >

  • </html>.- Este elemento encierra todo el contenido de la página entera y, a veces, se le conoce como el elemento raíz (root element).

Texto con énfasis: < em >

Marcar con énfasis las partes importantes de un texto

Texto importante: <strong>

Resalta con especial énfasis las partes más importantes de un texto.

Ejemplo.-

<p> <em>El dinero</em>es importante pero <strong>la salud</strong> lo es más. </p>

Resultado

El dinero es importante pero la salud lo es más.

Texto resaltado: <mark>

Representa un texto marcado o resaltado como referencia o anotación

Ejemplo.-

<p> El telescopio espacial <mark> Hubble </mark> de la NASA ha sacado muchas imágenes asombrosas. Una de las más memorables es el mosaico del perfil del Sombrero. Con su magnitud relativamente alta y a una distancia de <mark> 28 millones de años luz </mark> de la Tierra, Messier 104, nombre formal de Sombrero, se puede ver fácilmente por un telescopio. </p>

El telescopio espacial Hubble de la NASA ha sacado muchas imágenes asombrosas. Una de las más memorables es el mosaico del perfil del Sombrero. Con su magnitud relativamente alta y a una distancia de 28 millones de años luz de la Tierra, Messier 104, nombre formal de Sombrero, se puede ver fácilmente por un telescopio.

Texto de código

    }

</code>

Resultado

h1{

}

Etiquetas de listas

Listas ordenadas: < ol > ,< li >

Estas etiquetas permiten agregar texto en formato de lista con una prioridad en el orden de aparición de los elementos.

Ejemplo.-

    Equipos con más títulos de champions
    <ol>
        <li>Real Madrid</li>
        <li>Milan</li>
        <li>Liverpool</li>
    </ol>

Resultado

  2. Milan
  3. Liverpool

Listas desordenadas: < ul > ,< li >

Estas etiquetas permiten agregar texto en formato de lista sin una prioridad en el orden de aparición de los elementos.

Ejemplo.-

    <ul>
        <li>Manzanas</li>
        <li>Peras</li>
        <li>Sandias</li>
    </ul>

Resultado

  • Manzanas
  • Peras
  • Sandias

Tablas

Contenedor de la tabla: < table >

Esta etiqueta encierra todo el contenido de una tabla

Filas: < tr >

Agrega una fila a la tabla

Celdas de encabezado: < th >

Esta etiqueta permite agregar celdas para los encabezados de columna o fila

Datos: < td >

Esta etiqueta permite agregar los diferentes datos dentro de la tabla, deben estar dentro de la etiqueta <tr >

Ejemplo.-

Pais Habitantes Territorio Idioma
México 198.9 Español
Brasil 212.6 8516 Portugues
Canada 38.01 9985 Ingles/Frances

Es posible tambien crear celdas que ocupen más de una columna o más de una fila, para ello se usan los atributos colspan y rowspan. Si se busca que una celda se expanda sobre varias columnas se emplea colspan con el valor del numero de columnas que ocupara dicha celda, es decir si buscamos se expanda 2 columnas se agregara el atributo colspan="2".El atributo rowspan se usa de forma similar, se coloca en la celda que ocupara mas de una fila indicando en el valor el numero de filas.

Etiquetas multimedia

Imágenes

Permite agregar imágenes, debe incluir el atributo obligatorio src cuyo valor corresponde a la ruta donde se encuentra la imágen.

Ejemplo.-

<img src="img/esme.webp" alt="cascada esmeralda, oaxaca">

Resultado

Video

Permite agregar video directamente desde el navegador, agregada en la versión de HTML5

Ejemplo.-

<video src="video/big_buck_bunny.ogv" controls> </video>

Atributo Valor Descripción
src Dirección URL Video a reproducir. Obligatoria si actua como etiqueta contenedora.
poster Dirección URL de la imagen Muestra una imagen a modo de presentación.
autoplay Booleano Comienza a reproducir el video automáticamente.
loop Booleano Vuelve a iniciar el video cuando finaliza su reproducción (bucle).
muted Booleano Establece el video sin sonido (silenciado).
controls Booleano Muestra los controles de reproducción. Por defecto no se muestran.
width tamaño Indica el tamaño de ancho del video.
height tamaño Indica el tamaño de alto del video.

La etiqueta <video> también puede actuar como etiqueta contenedora e incluir varias etiquetas HTML para dotar de mayor compatibilidad, o capacidades adicionales.

Video con alternativas: <source >

Si utilizamos la etiqueta <video> como etiqueta contenedora, podemos incluir etiquetas <source > en su interior para proporcionar formatos alternativos y tener mayor compatibilidad con otros navegadores y navegadores antiguos que no soporten HTML5:

Ejemplo


    <video controls>
        <source src="video/big_buck_bunny.webm" type="video/webm">
        <source src="video/big_buck_bunny.ogv" type="video/ogg">
        <source src="video/big_buck_bunny.mp4" type="video/mp4">
        <img src="imagen.png" alt="Video no soportado" />
        Lo lamento; su navegador no soporta video HTML5.
    </video>
